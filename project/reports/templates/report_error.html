Error: No hay técnicos disponibles

Actualmente, no hay técnicos disponibles en el sistema. Puedes crear técnicos utilizando el siguiente comando:

docker-compose exec web python manage.py create_random_technicians `numero de técnicos`

Después de ejecutar este comando, podrás recargar la página y ver los nuevos técnicos en el sistema.

También tienes la opción de generar pedidos aleatorios directamente. Si no hay técnicos disponibles, el programa te preguntará si deseas crearlos antes de generar los pedidos. Utiliza el siguiente comando:

docker-compose exec web python manage.py generar_pedidos

Después de ejecutar este comando, podrás recargar la página y ver los nuevos pedidos en el sistema.
